🤖 AI News Summarizer
A comprehensive AI-powered news summarization system that fetches and processes the latest technology news using advanced search APIs.
📋 Project Overview
This project is part of a team development effort to create an intelligent news summarization application. The system uses Tavily API for web search and provides structured, categorized news summaries.
✨ Features

Multi-Category News Fetching: AI, Technology, Business, Crypto, Cybersecurity
Custom Topic Search: Search for any specific topic
Trending Topics Detection: Get the latest trending technology news
Interactive CLI Interface: User-friendly command-line interaction
Automated Report Generation: Comprehensive formatted reports
File Export: Save reports in text format
Configurable Parameters: Customize results count and time range

🚀 Installation
Prerequisites

Python 3.8 or higher
pip package manager
Tavily API key

Setup Steps

Clone the repository
bashgit clone <your-repo-url>
cd AiNewsSummary

Create virtual environment
bashpython -m venv venv

# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate

Install dependencies
bashpip install -r requirements.txt

Environment Configuration
Create a .env file in the root directory:
envTAVILY_API_KEY=your_tavily_api_key_here


💻 Usage
Interactive Mode (Recommended)
bashpython src/main.py -i
Command Line Options
Generate Category Report
bashpython src/main.py -c ai --save
python src/main.py -c tech -r 10 -d 5
Custom Topic Search
bashpython src/main.py -t "OpenAI ChatGPT" -r 5
python src/main.py -t "Tesla Cybertruck" --save
Comprehensive Report
bashpython src/main.py --comprehensive --save
python src/main.py --comprehensive --categories "ai,crypto,startup" -r 3
Trending Topics
bashpython src/main.py --trending -r 10

🚨 Troubleshooting
Common Issues

API Key Errors
Error: Tavily API key not found
Solution: Check .env file configuration

Network Timeouts
Error: Request timeout
Solution: Check internet connection, retry after few minutes

Empty Results
No articles found
Solution: Try different search terms or increase days parameter
